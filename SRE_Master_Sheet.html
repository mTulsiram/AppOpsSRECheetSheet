3. SQL Deep-Dive: Reads & Joins
SELECT * FROM users LIMIT 10;Basic read.
SELECT * FROM users ORDER BY id LIMIT 1 OFFSET 14;"15th User" Query (Skip 14, Take 1).
SELECT * FROM t1 UNION SELECT * FROM t2;Combines rows; REMOVES duplicates.
SELECT * FROM t1 UNION ALL ...Combines rows; KEEPS duplicates.
INNER JOIN:SELECT * FROM u JOIN o ON u.id=o.uid; (Matches in BOTH).
LEFT JOIN:SELECT * FROM u LEFT JOIN o ON u.id=o.uid; (Finds Orphans).
RIGHT JOIN:Keeps all Right table rows (Rarely used).
FULL OUTER JOIN:SELECT * FROM t1 FULL OUTER JOIN t2 ... (Keep All).
EXISTS:SELECT * FROM u WHERE EXISTS (SELECT 1...); (Fast Filter).
PK vs Unique:PK=Unique+NotNull. Unique=Allows 1 Null.
4. SQL Deep-Dive: DDL, DML & Perf
DROP TABLE old_logs;DDL: Deletes table structure permanently.
DELETE FROM users WHERE id=15;DML: Removes specific data row.
